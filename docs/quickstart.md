Getting started
===============

Requirements
------------

Python 3.5 or above.

* PyQt5 version?
* pyqtgraph 0.10.0 or above
* python serial 3.0 or above
* edfrw

On Debian and friends, running as su the following should do the job:

`apt-get install python3-pyqtgraph python3-pyqt5 python3-serial`
